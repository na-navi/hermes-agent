# na-navi/hermes-agent

Windows-native fork of [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent).

## What's different

- **se integration**: Everything + migemo as primary local search
- **PowerShell-first**: Shell abstraction via args (bash default, PS planned)
- **CodeGraph**: Under evaluation for codebase understanding

## Setup

```bash
git clone https://github.com/na-navi/hermes-agent.git
cd hermes-agent
# follow original install instructions (see README.original.md)
```
